## Сноска в Markdown

Markdown широко используется в разработке для создания документации[^1]

[^1]: Например, на GitHub, в Jupyter Notebooks и различных системах документации.

## Alert-блоки (Admonitions)

> [!TIP] > **Совет по использованию Markdown:**
> Для сложных документов комбинируйте Markdown с HTML-элементами для достижения лучших результатов.

## Escape-символы

Иногда нужно отобразить специальные символы Markdown как обычный текст:

- Чтобы показать решетку без создания заголовка: \# это не заголовок

## HTML в Markdown

<p style="color: blue; font-weight: bold; padding: 10px; border-left: 3px solid #2196F3;">
Этот текст оформлен с помощью HTML-тега &lt;p&gt; с CSS-атрибутами прямо в Markdown-документе. Такое сочетание позволяет создавать более сложное оформление, когда стандартных возможностей Markdown недостаточно.
</p>
