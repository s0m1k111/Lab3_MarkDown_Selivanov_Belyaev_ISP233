Markdown - это простой язык разметки для быстрого формирования текста (заголовок, списки, жирный/курсив), который легко преобразуется в HTML, и используется для создания документации и т.д
***
Основные возможности Markdown - это быстрое и простое формирования текста(заголовки, списки, ссылки, выделение), сохраняющее читабельность, для последующего преобразования в HTML или другие форматы, идеальное для документации и т.д.
***
Пример Формирования Заголовки:
# KKK
## KKK
### KKK
